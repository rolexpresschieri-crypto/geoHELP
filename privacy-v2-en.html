2. Service description

After registration, geoHELP lets you send help requests by SMS with GPS location and essential data to numbers configured for the service (operations centre / volunteers). Manual requests (Help) are prepared through the phone’s Messages app (mobile network / SIM). Optional MAN DOWN may detect falls or impacts and, after a cancellable 60-second countdown, send automatic SMS: first via the user’s SIM; if sending fails, via cloud provider Twilio (sender ID geoHELP). The service does not replace official emergency numbers (e.g. 112).
