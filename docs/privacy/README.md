# Privacy policy (GitHub Pages)

Pubblicare su GitHub Pages (cartella `/docs` del repo):

- **IT (Play Console, Italia):** `…/privacy/privacy-v2-it.html`
- **EN:** `…/privacy/privacy-v2-en.html`
- **Indice:** `…/privacy/` o `…/privacy/index.html`

Dopo l’abilitazione Pages, URL tipo:

`https://<utente>.github.io/<repo>/privacy/privacy-v2-it.html`

Aggiornare i file copiando da `app/src/main/assets/privacy/privacy_v2_*.html` quando cambia la versione in app.
